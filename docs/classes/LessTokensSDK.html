    Class LessTokensSDK

    Main LessTokens SDK class

    Provides a simple interface to compress prompts using LessTokens API and send them to various LLM providers (OpenAI, Anthropic, Google, DeepSeek).

    Example

    const sdk = new LessTokensSDK({
      apiKey: 'your-less-tokens-api-key',
      provider: 'openai',
    });

    const response = await sdk.processPrompt({
      prompt: 'Your prompt here',
      llmConfig: {
        apiKey: 'your-openai-api-key',
        model: 'gpt-4',
      },
    });

      Compress a prompt without sending to LLM

      Useful when you only want to compress a prompt without sending it to an LLM. Returns compression results including token counts and savings.

      Parameters

        The prompt to compress

      • Optionaloptions: CompressionOptions

        Optional compression settings

        Compression options

        • Optionalaggressive?: boolean

          Use aggressive compression (default: false)

        • OptionalpreserveContext?: boolean

          Preserve context during compression (default: true)

        • OptionaltargetRatio?: number

          Target compression ratio (0.0 to 1.0, default: 0.5)

      Returns Promise<CompressedPrompt>

      Promise resolving to compression results

      Throws

      If compression fails

      Example

      const result = await sdk.compressPrompt('Very long prompt...', {
        targetRatio: 0.3,
        aggressive: true,
      });

      console.log(`Compressed from ${result.originalTokens} to ${result.compressedTokens} tokens`);
      console.log(`Savings: ${result.savings} tokens (${result.ratio * 100}% ratio)`);

    processPrompt

    • processPrompt(options: ProcessPromptOptions): Promise<LLMResponse>

      Process a prompt through LessTokens compression and send to LLM

      This method:

      1. Compresses the prompt using LessTokens API
      2. Sends the compressed prompt to the configured LLM provider
      3. Returns the LLM response with compression metrics

      Parameters

      • options: ProcessPromptOptions

        Processing options

        Options for processing a prompt

        • OptionalcompressionOptions?: CompressionOptions

          Compression options

        • llmConfig: LLMConfig

          LLM API configuration - supports all provider-specific options

        • OptionalmessageContent?: string | ((compressed: CompressedPrompt) => string)

          Custom message content. Can be a string or a function that receives the compressed prompt and returns the content

        • OptionalmessageRole?: string

          Custom message role (default: 'user')

        • Optionalmessages?: { content: string; role: string }[]

          Additional messages to include in the conversation (for multi-turn conversations)

        • prompt: string

          Prompt text to be processed

      Returns Promise<LLMResponse>

      Promise resolving to LLM response with compression metrics

      Throws

      If compression or LLM request fails

      Example

      // Basic usage
      const response = await sdk.processPrompt({
        prompt: 'Explain quantum computing',
        llmConfig: {
          apiKey: process.env.OPENAI_API_KEY!,
          model: 'gpt-4',
          temperature: 0.7,
        },
      });

      // Custom role
      const response2 = await sdk.processPrompt({
        prompt: 'Explain quantum computing',
        llmConfig: { apiKey: '...', model: 'gpt-4' },
        messageRole: 'system',
      });

      // Custom content with function
      const response3 = await sdk.processPrompt({
        prompt: 'Explain quantum computing',
        llmConfig: { apiKey: '...', model: 'gpt-4' },
        messageContent: (compressed) => `Compressed: ${compressed.compressed}\nOriginal tokens: ${compressed.originalTokens}`,
      });

      console.log(response.content);
      console.log(`Saved ${response.usage.savings}% tokens`);

      Process a prompt with streaming response

      Similar to processPrompt, but returns a stream of chunks instead of waiting for the complete response. The final chunk includes compression metrics.

      Parameters

      • options: ProcessPromptOptions

        Processing options

        Options for processing a prompt

        • OptionalcompressionOptions?: CompressionOptions

          Compression options

        • llmConfig: LLMConfig

          LLM API configuration - supports all provider-specific options

        • OptionalmessageContent?: string | ((compressed: CompressedPrompt) => string)

          Custom message content. Can be a string or a function that receives the compressed prompt and returns the content

        • OptionalmessageRole?: string

          Custom message role (default: 'user')

        • Optionalmessages?: { content: string; role: string }[]

          Additional messages to include in the conversation (for multi-turn conversations)

        • prompt: string

          Prompt text to be processed

      Returns Promise<AsyncIterable<StreamChunk, any, any>>

      Async iterable of stream chunks

      Throws

      If compression or LLM request fails

      Example

      const stream = await sdk.processPromptStream({
        prompt: 'Tell a story',
        llmConfig: {
          apiKey: process.env.OPENAI_API_KEY!,
          model: 'gpt-4',
        },
      });

      for await (const chunk of stream) {
        if (!chunk.done) {
          process.stdout.write(chunk.content);
        } else {
          console.log(`\nSaved ${chunk.usage?.savings}% tokens`);
        }
      }
